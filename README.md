# UrbanSeed 🌱

UrbanSeed es una herramienta web para planificación de cultivos urbanos que utiliza datos climáticos y recomendaciones de cultivo.

## Características
- Recomendaciones de cultivos según clima
- Planner de siembra
- Interfaz simple para usuarios urbanos

## Demo
https://zavalaby16.github.io/urbanseed/

## Tecnologías
- HTML
- CSS
- JavaScript
- GitHub Pages

## Objetivo del proyecto
Ayudar a personas a planificar cultivos urbanos de forma sencilla utilizando información ambiental.
